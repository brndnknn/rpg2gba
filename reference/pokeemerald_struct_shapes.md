# pokeemerald-expansion struct shapes (P4)

Verified 2026-05-20 against the fork at `/home/b/repos/pokeemerald-expansion`,
shallow clone HEAD `21c24202`. This is the authoritative reference for the
Phase 2 C emitters. If the fork is updated, re-verify and update this doc
**before** changing an emitter (PHASE2_PLAN.md P4 rule).

All data tables in this expansion use **C99 designated initializers**, so the
emitters only need to write the fields we have Uranium data for; every other
field takes its struct default. This is what makes best-effort emit safe.

---

## Species — `struct SpeciesInfo` (include/pokemon.h)

Data files: `src/data/pokemon/species_info/gen_N_families.h`, all `#include`d
by `src/data/pokemon/species_info.h`. Keyed `[SPECIES_X] = { .field = ..., }`.

### Field mapping (Uranium dexdata.dat → expansion)

| Uranium field | expansion field | notes |
|---|---|---|
| BaseStats[0..5] HP/Atk/Def/Spd/SpA/SpD | `.baseHP .baseAttack .baseDefense .baseSpeed .baseSpAttack .baseSpDefense` | u8 each |
| Type1, Type2 | `.types = MON_TYPES(TYPE_A, TYPE_B)` | macro defaults Type2→Type1 if same; map Uranium type index via `type_internal_names.json` |
| Rareness | `.catchRate` | u8 |
| BaseEXP | `.expYield` | u16 |
| EffortPoints[0..5] | `.evYield_HP .evYield_Attack .evYield_Defense .evYield_Speed .evYield_SpAttack .evYield_SpDefense` | each a **2-bit** bitfield → max value 3; emit only nonzero |
| GenderRate (enum) | `.genderRatio = PERCENT_FEMALE(x)` / `MON_GENDERLESS` | `PERCENT_FEMALE(p) = min(254,(p*255)/100)`; genderless = `0xFF` |
| StepsToHatch | `.eggCycles` | **convert:** cycles ≈ round(steps/256) |
| Happiness | `.friendship` | u8; vanilla default `STANDARD_FRIENDSHIP` |
| GrowthRate (enum) | `.growthRate = GROWTH_*` | see enum below |
| Compatibility[0..1] (egg groups) | `.eggGroups = MON_EGG_GROUPS(EGG_GROUP_A, EGG_GROUP_B)` | see enum below |
| Abilities[0..1] + HiddenAbility[0..3] | `.abilities = { ABILITY_A, ABILITY_B, ABILITY_HIDDEN }` | `NUM_ABILITY_SLOTS = 3` (2 normal + **1** hidden); see drift note |
| Color (enum) | `.bodyColor = BODY_COLOR_*` | see enum below |
| Height | `.height` | decimeters (Uranium compiled value already ×10) |
| Weight | `.weight` | hectograms (already ×10) |
| — (messages.dat) | `.speciesName = _("..")` | from `species_names.json` |
| — (messages.dat) | `.categoryName = _("..")` | from `species_kinds.json` (13-byte field) |
| — (messages.dat) | `.description = COMPOUND_STRING("..")` | from `species_pokedex.json` |
| attacksRS.dat | `.levelUpLearnset = sXLevelUpLearnset` | see below |
| eggEmerald.dat | `.eggMoveLearnset = sXEggMoveLearnset` | see below |
| evolutions.dat | `.evolutions = EVOLUTION(...)` | see below |

`.natDexNum` — Uranium has no national dex; leave default (NONE). Tandor dex
number (regionals.dat) carried in a sidecar for Phase 5.

`.teachableLearnset` — **do NOT emit.** Build-generated by
`tools/learnset_helpers/make_teachables.py` from `all_learnables.json` +
TM/HM/tutor JSON. §2.5 feeds that generator instead of writing arrays.

> **Drift — hidden abilities.** Expansion = 1 hidden slot. Uranium ships up to
> 4. Best-effort: first hidden → `.abilities[2]`; full list → sidecar +
> `needs_engine`. Surface at V5 gate.

## Constants — `include/constants/species.h`

Plain `#define SPECIES_X  N` lines, one per species, `SPECIES_NONE 0` first.
Names follow Pokémon Showdown conventions for vanilla mons; Uranium-originals
get `SPECIES_<INTERNAL_NAME>`.

## Level-up learnsets

File: `src/data/pokemon/level_up_learnsets/gen_N.h`.

```c
#define LEVEL_UP_MOVE(lvl, moveLearned) {.move = moveLearned, .level = lvl}
#define LEVEL_UP_END {.move = LEVEL_UP_MOVE_END, .level = 0}

static const struct LevelUpMove sBulbasaurLevelUpLearnset[] = {
    LEVEL_UP_MOVE( 1, MOVE_TACKLE),
    ...
    LEVEL_UP_END
};
```
`struct LevelUpMove` (include/pokemon.h ~616): `{ u16 move; u16 level; }`-ish
(use the macros, don't hand-pack).

## Evolutions

```c
struct Evolution { u16 method; u16 param; u16 targetSpecies;
                   const struct EvolutionParam *params; };

// in species entry:
.evolutions = EVOLUTION({EVO_LEVEL, 16, SPECIES_IVYSAUR}),
// EVOLUTION(...) macro appends {EVOLUTIONS_END} (=0xFFFF) automatically.
```
Methods (include/constants/pokemon.h): `EVO_NONE, EVO_LEVEL, EVO_TRADE,
EVO_ITEM, EVO_SPLIT_FROM_EVO, EVO_SCRIPT_TRIGGER, EVO_LEVEL_BATTLE_ONLY,
EVO_BATTLE_END, EVO_SPIN`. Map Uranium evolution-method bytes → these before
emit (Uranium has more methods; unmapped → `needs_engine` + comment).
