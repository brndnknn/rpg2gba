## Key File Notes

- `scripts/extract_rgssad.py` ‚Äî pure-stdlib RGSSAD v1 extractor. Run it once per Uranium release to unpack `Uranium.rgssad` ‚Üí `<dir>/_unpacked/`. Algorithm: XOR stream cipher, initial key 0xDEADCAFE, advance `key = key * 7 + 3`. File data uses the global key as starting seed and advances per 4-byte group within the file; the global key is *not* advanced by file data, only by name and size fields.
- `scripts/recon_maps.rb` ‚Äî Marshal stubs for RMXP classes. Critical: `RPG::Event::Page` (and its `Condition`/`Graphic`) must be **nested** inside `RPG::Event`, not flat. `Color`/`Tone`/`Table` use the user-defined `_dump`/`_load` *class-method* pattern, not `marshal_dump`/`marshal_load` instance methods. Both gotchas were live bugs we hit on first run.
- `scripts/recon_scripts.rb` ‚Äî section name normalization handles spaces, `*`, leading `_`, divider strings (`=====`); use `vanilla?(name)` rather than direct allowlist lookup. Inflated source is transcoded `Windows-1252 ‚Üí UTF-8` (with replacement) and written via `File.binwrite` ‚Äî the prior `File.write(..., encoding: 'utf-8')` raised on 85 of 260 sections leaving 0-byte files. If a future change reintroduces UTF-8-strict writes, ~85 sections will silently fail to dump.

## Uranium-Specific Discoveries

- **Essentials version: v17.** Evidenced by 29 sections suffixed `_v17` in `Scripts.rxdata`. See `reference/uranium_essentials_version.md`.
- **No PBS source files ship.** Only Essentials' compiled `.dat` files are distributed ‚Äî see `reference/uranium_dat_inventory.md`.
- **No `pokemon.dat`/`abilities.dat` despite Essentials convention.** Species data lives somewhere else in Uranium's Data/ ‚Äî candidates are `attacksRS.dat`, `tmpbs.dat`, or inside `dexdata.dat`. **Open question ‚Äî needs spike before Phase 2.**
- **Map/event scale:** 199 maps, 5,301 events, 8,429 event pages. 132/199 maps have at least one page with ‚â•30 commands.
- **RMXP event trigger codes (got this wrong once):** `0` = Action Button (talk), `1` = Player Touch, `2` = Event Touch, `3` = Autorun, `4` = Parallel. (Map200's step-on doormat warp is trigger 1, confirming.) The talk-once NPC idiom = a 2-page event, base page trigger 0 sets self-switch A (code `123 [A,0]`), a later page gated on `condition.self_switch_valid`+`self_switch_ch`. **860** events set a self-switch via 123; **999** have a self-switch-gated page. Native global `Control Switches` (121, in-list `if`-branch) is rare (539) ‚Äî the clean single-page global-switch talk-twice NPC essentially doesn't exist in Uranium.
- **Temp-switches are a SECOND, distinct self-switch-like idiom (`setTempSwitchOn`/`tsOn?`, 345√ó across 77 maps).** Defined in `022_Game_Event_v17.rb` (`@tempSwitches[c]=true`); `@tempSwitches` is init'd `{}` in `Game_Event#initialize` and is **NOT** in `$PokemonGlobal` (unlike `eventvars`, which IS persisted) ‚Üí temp-switches are **per-map-visit runtime state that resets on map reload** = pokeemerald **TEMP flag** semantics (0x0‚Äì0x1F, auto-reset on warp), **not** persistent self-switches. Set/cleared via script call (code 355: `setTempSwitchOn("A")`/`setTempSwitchOff`), checked via `tsOn?/tsOff?` (or page conditions). **Conversion gap:** the agent maps `setTempSwitchOn("A")`‚Üí`setflag(FLAG_MAP..SSA)` (treats it as a saved self-switch ‚Äî both wrong on persistence AND undefined, since orchestrator minting only covers code 123 + `self_switch_valid` pages). Disposition is a ¬ß9 #2 gate decision (TEMP-flag map vs saved self-switch vs queue). `pbSetSelfSwitch` (059/221) is the REAL persistent self-switch (‚Üí`$game_self_switches`); code 123 is its RMXP-native form (Layer 1 mints those).
- **`MapInfos` editor names are unreliable as in-game identity (Phase 5 concern).** Discovered at the Phase 3 V4 wiki spot-check: map 7's `MapInfos` name is "Comet Cave" but its content is the wiki's **Passage Cave** (BGM "PU-Passage Cave", `pbPushThisBoulder` Strength puzzle, the early cave's connection pattern). Phase 5 (overworld wiring / display names) must derive location identity from content + the town-map / wiki, **not** from `MapInfos`/`map_names.json`.
