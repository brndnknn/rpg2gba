# rpg2gba

A pipeline for converting RPG Maker XP / Pokémon Essentials fan games into playable GBA ROMs, using Pokémon Uranium as the first end-to-end test case.

The pipeline tooling is the long-term deliverable. The Uranium ROM is the proof of concept.

**Personal use only.** The resulting ROM incorporates copyrighted assets and is not for distribution.
